# AzulTracker

A full-stack gym workout tracking application built with ASP.NET Core (.NET 10), React + TypeScript, and Azure SQL.

## Tech Stack

- **Backend:** ASP.NET Core Web API (.NET 10)
- **Frontend:** React + TypeScript (Vite)
- **Database:** Azure SQL + Entity Framework Core
- **Auth:** JWT
- **Cloud:** Azure App Service, Azure SQL, Key Vault, App Insights
- **CI/CD:** Azure DevOps
